    • Research

  • Class-Based Summarizations of Chinese Microblogs
    • Supervisor: Hua Xu, Associate Professor, Tsinghua National Laboratory for Information Science and Technology.
    • March 2014 - Now
    • In times of big data, there are millions of Chinese microblogs(also known as weibo) posted every day. It is necessary to generate summarizations of these microblogs automatically. However, many developed systems and algorithms like The Phrase Reinforcement and Hybrid TF-IDF whose target language is English performs poorly in Chinese corpus, because much more different or informal expression in weibo delivering the same meaning weaken the power of words-cooccurence-based algorithms. This research focuses on designing a class-based algorithm to generate summaries of Chinese microblogs. It firstly uses machine learning algorithms such as LogicLDA and Word2Vec to distribute words into classifications in which words deliver the same meaning. Then we use a tree pattern reinforcement algorithms to generate summaries based on the classification in the first step. Related work and algorithm references can be seen in this slide.

    Major Projects

  • Dynamic Academic Hot Topic Discovery
    • April 2014 - June 2014, with Yang Liu
    • In this project, we build a web service which offers a convenient way to detect hot academic topics and their relevance over time. We provide two APIs for users to query, one is to present related topics in a quantity way of the input over time, the other is to draw a chart line showing the dynamic relevance of two academic topics users have input. We take advantage of the algorithm Word2Vec to train a model to discover the relative academic hot words at each year according to database of NIPS. The web service is built under the framework of python flask with several javascript libraries like ichartjs. More information and demo can be seen in this poster.
  • An Omnipotent Card Game Center
    • September 2013 - December 2013, as a member of a developing group of five named Colder
    • In this software project, we develop an on-line game center system where users can upload rule files to create a kind of new game based on cards. We implement this platform with golang programming language and revel framework. My contribution to this project is to design a language which combines C-style function and regular expression to describe any card-based games and implement an interpretor of it using the knowledge of compiler and formal language. Here offers a brief introduction to this language and the structure of modules of the whole project.
  • A Simple Rhythm Master Game Platform
    • June 2013, with Yang Liu
    • We use a 8*16 LED matrix, CPLD with only 240 logic unit plus few independent digital circuits to build a simple platform on which we can play rhythm games. We take LED as screen and use VHDL as programming language. The platform generates arrows directing up or down randomly and moving at a certain speed from right to left. The speed can be adjusted according to difficulty levels. The player should click the right key which is consistent with the arrow's direction when the arrow is in the hitting zone. When playing this game, we rhythmically click the up-key and down-key, so we call it 'rhythm master game'. More information and demos are offered here.
